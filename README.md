# Beans for a Living Soil - Play

Distribuição pública e autônoma do protótipo jogável **Beans for a Living Soil**.

[Jogar no navegador](https://marcoantoniobarreiros.github.io/beans-for-a-living-soil-play/)

O jogo é executado diretamente no navegador por GitHub Pages. Este repositório contém somente o artefato HTML compilado; o código-fonte modular, os testes e a documentação de desenvolvimento permanecem em um repositório privado.

## Controles

- Movimento: `A`/`D` ou setas direcionais.
- Salto: `W`, seta para cima ou `Espaço`.
- Dash: `Shift`.
- Pulso mineral: `E`.

As habilidades adicionais são liberadas durante a progressão da fase.
